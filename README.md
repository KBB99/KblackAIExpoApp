# KblackAIExpoApp
## Get Started

### Clone
```bash
git clone https://github.com/KBB99/KblackAIExpoApp/
```

### Enter Directory
```bash
cd KblacAIExpoApp
```

### Install Dependencies
```bash
yarn
```

### Start Bundler
```bash
yarn start
```

### Run iOS
```bash
i
```

### Run Android
```bash
a
```
